# Drawer app

Eenvoudige demo van hoe je in flutter een drawer aan je scherm kunt toevoegen.

We gebruiken hiervoor hetzelfde scherm als voor de [katten demo](../service_app/lib/katten.dart), maar nu hebben we aan de linkerbovenkant een hamburger-menu waarop je kunt klikken om extra informatie over de app te krijgen. Let op: dit hamburger-menu zit op dezelfde plek als de `appBar` in de `Scaffold`, dus als je die niet hebt dan krijg je 'm niet te zien.
